Win10 软件配置方案

发表于 2019-05-05 | 分类于 工具 | 阅读次数：

电脑，是生产力工具。因此软件配置的目的，是最大化生产力，是在最短的时间，使用上最合适的工具。

1. 基础设置

如果是第一次开机装软件：

  1. 按照指示设置 Windows，包括地区（中国）、语言、输入法（可添加英文输入法及键盘）等。从“所有设置→系统→关于查看”电脑的基本信息；
  2. 由于电脑可能较早就安装了 Windows 系统，而开机的时候已经过了一段时间，因此需要对 Windows 系统进行更新。“所有设置→更新和安全→Windows 更新→检查更新”
  3. 使用 Win10 自带的 Edge 浏览器，使用 bing.com 国际版进行搜索，下载软件及安装。此时最主要的是需要能对软件清理及管理，下载软件及安装。此时最主要的是需要软件清理及管理)的软件，安全防护软件，以及解压软件 ；

如果是想对已有的系统进行优化：可参考以下的软件推荐及配置方法。

2. 系统必备软件

360 Total Security 或火绒安全：杀毒软件。

说明：虽说Windows自带的Defender也不错，不过一些专业杀毒软件的优化做得更好些。相比国产360，出口的360 total security 弹窗少，安静，所以可以用。火绒也是被很多人推荐，不扰人，功能较为齐全。火绒安全的主要功能是主动防护而非主动杀毒。

配置：无，开箱即用。

缺点：资源占用率稍高。

CCleaner/Defraggler/Recuva：梨子公司三件套。

说明：梨子公司三件套是Windows必备软件。

  • CCleaner 用于清理电脑垃圾，卸载软件，管理开机启动项等等；
  • Defraggler 用于清理电脑里面的碎片；
  • Recuva 用于恢复误删的文件。

配置：

  • 无，都是开箱即用。
  • 小技巧，Defraggler 在扫描完碎片后，先查看文件，按大小排序，全选后把大碎片勾除，把剩下的中小碎片清理完，最后再解决大碎片。因为其中几个大碎片是无法清理的，全部一起清理会造成很多小碎片没有完整清理。

缺点：非付费用户需要手动升级。

Everything：快速搜索。

说明：快速且强大的搜索，体型小巧，是 Windows 必装软件。

配置：无，可保存搜索的语句。

基本语法：

  • file:XXX, folder:XXX，指定搜文件、文件夹；
  • XX YY, XX|YY, XX!YY：逻辑运算符，与或非；
  • ext:, case:, size:：指定拓展名、大小写，文件大小；
  • da:, dc:, dm:：指定文件读取、创建和修改的日期；

缺点：无明显缺点，除了精确搜索时需要掌握一些技巧（如正则表达式，逻辑运算符），有一点入门门槛。

好压：压缩解压软件。

说明：很多安装包都是压缩文件，因此解压软件是必备的。好压功能完善，比 7z 好用，但有广告弹窗；

配置：注意安装选项去除额外的安装，并使用任务管理器跟踪弹窗的来源进行删除；

缺点：弹窗，丑。

Memreduct：降低内存。

说明：若机子内存不高，启动太多应用时会占用大量内存，使用 Memreduct 可清理掉部分内存，腾出空间；

配置：设置开机启动和最小化启动，跳过警告，设置自动清理内存的阈值（触发器），设置快捷键（Ctrl+`），设置”使用透明背景“，设置鼠标控制（双击为清理内存）；

缺点：若超过设定清理内存的阈值，会一直不断清理并弹窗，有些烦人。

3. 系统界面强化软件

在系统强化上，主要是做到界面清爽，所需的工具和资料能随手可得或可查，具体目标如下：

  • 桌面无图标，保持清爽；
  • 任务栏放置常用的软件，可随时使用；
  • 需要的软件可随时呼叫出现；
  • 资料可以随时方便的查找。

Launchy：快速启动。

说明：

  • 桌面最好一个软件图标都不要放，那打开软件就需要快速启动软件的帮忙，Launchy 就是其中的佼佼者。
  • Launchy 在 Win10 下简单易操作，使用特定皮肤后颜值高。Autohotkey 门槛较高，需要记得快捷键，不太人性化。AltRun 一直没更新，颜值低。

配置：

  • 下载苹果主题 （OS X Yosemite） ，解压后放在launchy/skins文件夹里面，之后在 Launchy 的“选项”——“皮肤”里面就可以选择了；
  • 设置呼出的快捷键。右键点击 Launchy，“选项”——“常规”——“Hotkey”，为了不与快速关闭程序的快捷键Alt+Space+C冲突，我把 Launchy 的快捷键设置为Ctrl+Space；
  • 一般来说 Launchy 会扫描软件，然后输入软件名称就可以直接打开软件。但是有些扫描不到的软件，就需要在选项 - 插件 - Runner那里添加。直接把软件 exe 拖进去即可，还可以修改呼出的名字；
  • Launchy 和 Everything 可集成起来，这样可以直接快捷键呼出 Lanuchy 后直接输入搜索命令，极为方便。为了让 Launchy 和 Everything 集成，需要在 Runner 那里把 Everything 的 exe 拖进去（此时 Program 那里应该是C:\Program Files\Everything\Everything.exe），然后把关键词（Name）改成自己喜欢的，我用的是f，最后把 Arguments 改为-s " $$即可。使用的时候是快捷键呼出 Launchy，然后输入 f，按 Tab 键，再输入要搜索的内容，回车即可；
  • 要使用网络搜索引擎时，可以在“选项”——“Weby”插件那里设置。比如设置百度搜索，https://www.baidu.com/s?ie=UTF-8&tn=null&wd=%1。使用时需要呼出 Launchy，输入搜索引擎关键字如 Google，然后按 Tab 键，输入搜索内容，回车即可；
  • Launchy 可设置开机启动。“Win+R 打开运行窗口→输入 shell:startup→打开启动文件夹→把 Launchy 的快捷方式加进去即可”。

缺点：时不时不明原因的崩溃，然后皮肤、快捷键、Runner 那里的软件都要重设。

Free Launch Bar：替代 Win10 原生的任务栏。

说明：Free Launch Bar 可以完全替代 windows 原生的任务栏，并且可以把任务放在桌面底部中间，非常美观。

配置：

  • 安装 FreeLaunchBar 之后，右键点击任务栏，在“工具栏”那里选择“Free Launch Bar”，然后把“锁定工具栏”的勾去掉，这时就可以把任务栏随意移动了，调到合适的位置（我是调到中间），再把“锁定工具栏”的勾点上即可；
  • Win10 和白色图标更配。方法是首先下载白色图标（可以在 iconfont下载图标，下载时把颜色设为白色，下载 png 后使用在线 png 转 ico，把图标转成 ico 格式即可）。然后把软件的快捷方式放到桌面，右键快捷方式，在属性中更换图标。最后把更换后的图标固定在快速启动栏上，再删除快捷方式即可；
  • 去除桌面“我的电脑”和“回收站”图标方法：“设置→个性化→主题→桌面图标设置→取消勾选我的电脑和回收站即可”。

缺点：图标处理稍微麻烦。

TranslucentTB：让任务栏变透明。

说明：可以让任务栏变透明，美观度大幅上涨。

配置：

  • 安装完成后，右键点击图标，Regular→Clear，即可让任务栏透明；
  • 勾选 Open at boot，设置开机自动启动。

缺点：开机后有一小段时间的运行程序才能启动，但无伤大雅。

QTTabbar：资源管理器多标签。

说明：QTTabbar 和 Clover 都能实现资源管理器多标签功能，但 QTTabbar 相比 Clover 更小巧，bug 比 Clover 少非常多。Clover 经常不明原因崩溃，而且在 Win10 下表现并不好。

配置：调整 Option 里面的选项，如双击标签是关闭，右键空白处关闭 QTTabs standard buttons；

缺点：配置比较麻烦。

Smart Taskbar：智能任务栏。

说明：其主要功能就是在窗口最大化的时候自动隐藏任务栏，然后平时显示任务栏。其实可以直接在 Windows 设置里面调整自动隐藏任务栏，但在没有最大化的窗口时确实是不需要隐藏任务栏的，因此这个软件是可以安装的；

配置：可将任务栏图标设置为”小按钮“，也可以设置为自动隐藏（即”智能“）；

缺点：有一些 bug 影响体验，如在窗口最大化时，有时候鼠标移动到任务栏时任务栏会一直隐藏，不弹出来。使用 FoxitReader 最大化的时候会循环一直最大化。

4. 日常所需软件

Chrome：网络浏览器。

说明：浏览器还是喜欢 Chrome，各式好用的插件，F12 网页调试神器。

配置：主要插件列表。

  • Momentum：强化并美观的新标签页。设置 show links, search, weather；
  • Extension Manager：一键管理所有扩展；
  • AdBlock：屏蔽广告；
  • Proxy Switchy Omega：翻墙，配合 XXNet 使用；
  • OneTab：将一系列网页压缩为一个标签页，下次打开浏览器时会自动出现，即稍后再读；
  • The Great Suspender：网页在设定时间内未浏览则自动设为暂时关闭状态，以减少 Chrome 对内存的消耗；
  • Ghostery：隐私广告拦截；
  • Vimuim：纯键盘操作 Chrome；
  • Chromium Wheel Smooth Scoller：更顺畅的滑动条；
  • 迅雷下载：用于启动迅雷下载；
  • 简悦：专注模式的网页阅读；
  • Markdown Here：用 Markdown 写邮件；
  • IE Tab：在 Chrome 中使用 IE 内核；
  • 购物党自动比价工具：观察电商网站商品价格走势，领取淘宝卖家券；
  • WizClipper：为知笔记剪藏器；
  • Mendeley Importer：Mendeley 文献导入器；
  • Xpath Helper：用于测试爬虫；
  • 划词翻译：网页上划词即可翻译，有国内翻译源，较为稳定；
  • Octotree：侧边栏显示 Github 项目的树状结构；
  • 京价保：自动登陆京东领取京豆以及监测已买产品的价格变动完成价格保护；

两个额外增强型扩展：

  • Stylus：重设网页布局，可以重设 Wikipedia、百度、淘宝、微博等网页的布局；
    使用 Stylus 需要在安装后，在官方网站 userstyles 中下载布局，优秀的布局有：
    • 百度轻
    • Wikipedia New Light
    • 淘宝助手
    • Weibo V6 Patch
    • 淘宝登录页
    • Youku Cleaner
    • 百度文库-屏蔽
    • 百度百科-屏蔽
    • 百度知道-屏蔽
    • 百度经验-屏蔽
  • Tampermonkey：油猴，可安装网页脚本做各种事情，打开网络资源的新大门；
    使用 Tampermonkey 需要安装之后，在脚本网站 Greasyfork 中下载脚本，优秀的脚本有：
    • 豆瓣资源下载大师：神器，下载豆瓣书籍、电影的资源大全；
    • 破解 VIP 会员视频集合：破解各个视频网站的 VIP 限制；
    • 百度网盘直链下载助手：直接获取百度网盘下载链接，但浏览器必须安装迅雷插件来直接下载；
    • 全网音乐一键免费下载：解析音乐下载地址，可直接下载各个平台的音乐；
    • 网易云音乐直接下载：更简捷的在网易云音乐的网站上下载音乐；
    • 京东自营过滤：过滤掉京东搜索结果中的非自营商家；
    • 京东热卖跳转：直接从京东热卖跳转到购买页面中；
    • 网页限制解除：解除掉网页的一些限制如不允许复制；
    • bilibili 下载：下载 B 站的视频；
    • 电影天堂直接提供迅雷下载链接：使用电影天堂时在电影页面直接显示迅雷下载链接；

缺点：Chrome 极为消耗资源。

XXNet：翻墙。

说明：免费翻墙，速度有保证，最多每天可以有12G的流量（!!!），Facebook, Twitter, Youtube 都可流畅访问。

配置：

  • 安装以及 Chrome 设置；
    • 下载并解压后，双击运行start.vbs，一路允许，然后程序会自动添加快捷方式；
    • 右键点击在通知栏的小图标，勾选“全局PAC智能代理”；
    • 打开 Chrome 浏览器，输入chrome://extensions进入扩展页面。打开XX-Net/SwitchyOmega文件夹，把里面的SwitchyOmega.crx拖进 Chrome 里面进行安装；
    • 在 SwitchyOmega 界面，“导入/导出”——“从备份文件恢复”——选择XX-Net/SwitchyOmega文件夹里面的OmegaOptions.bak文件；
    • 导入后，在 SwitchyOmega 界面，“情境模式”——“XX-Net自动切换”——“立即更新情景模式”；
    • 返回 Chrome 主页面，点击右上角的 SwitchyOmega 按钮，选择XX-Net自动切换。再右键点击在通知栏的小图标，勾选“取消全局代理”；
  • 创建 Google AppID，获得更多且更稳定的流量；
    • 打开https://console.developers.google.com，“顶部Project”——“创建项目”——随便输入名称然后点“创建”，即可。如果想最大化 XX-Net 的流量，可以最多申请 12 个 AppID；
    • 单击小图标，进入设置页面http://127.0.0.1:8085/，“部署服务端”——输入所有申请好的 AppID后（每个ID用|分隔）——“开始部署”——“允许”授权；
    • 部署完成后切换到设置页面的“配置”——输入所有 AppID（每个 ID 用|分隔）——“保存”；
    • 可以通过appengine.google.com查看流量使用情况；
  • 具体查看中文文档。

缺点：配置较为复杂，访问 Google 不是非常稳定。

Notepad++：文件查看。

说明：Notepad++ 小巧，可以打开多种格式的文件，查看文件时非常有效快捷。相对于强大的全键盘操作编辑器 Vim 和 Emacs，如果没有特别的效率要求，Notepad++ 已经足够用了。而相比于 UltraEdit，Notepad++ 功能较弱，但小巧且免费。

配置：如果只是用 Notepad++ 来查看文件，不安装插件，开箱即用就行。

缺点：颜值低。

IrfanView：图片浏览器。

说明：免费，小巧，功能完善。

配置：无特别设置，但翻到最后一张图片后默认会弹窗提示，有点不舒服。所以可以通过”Options”——“Properties/Settings”——“Browsing/Editing”——把”Show Browse dialog”改选为”Loop current folder”，即可。

缺点：颜值比较低，需要掌握一些快捷键。

为知笔记：知识管理。

说明：个人用过的体验最好的知识管理软件。可替代产品是有道云笔记。印象笔记由于灵活性低，软件大，资源消耗多，因此不作推荐，但依然有非常多忠诚用户。不过印象笔记的多设备同步是做得非常出色的。

配置：

  • 需要申请账号才能使用，也让云端可以同步笔记；
  • 需要在 Chrome 里面安装“为知笔记剪藏器”插件，就可以剪辑网页，收藏觉得好的文章；
  • 可以安装插件，但好像没有什么必装不可的插件；
  • 学习个人知识管理，这篇文章讲得很好；
  • 学习 Markdown 语法，为知笔记是极好的 Markdown 编辑器，依靠 Markdown 语法，用为知笔记可以格式写出很好的笔记。

缺点：资源消耗比较大，此外在数据安全性上，小公司能否一直运营是个疑问。

迅雷极速版：下载工具。

说明：迅雷的稳定性和速度依然非常好，可下载 bt，而且迅雷极速版则少了一些广告和弹窗。很多人推荐的神器 Internet Download Manager，我没有过所以不作阐述。

配置：无。

缺点：由于不再更新该产品，每次打开都会提示升级。

Motrix：百度网盘下载。

说明：现在由于对百度云上面的文件下载链接解析困难，所以现在用迅雷也很难下载百度云的文件了。现在的替代方案是使用 Motrix 下载器，该下载器与迅雷相似也可以下载 BT，并且可以通过下载和安装 BaiduExporter 插件，在百度云的页面形成 MO.app 按钮，点击即可直接用 Motrix 下载；

配置：下载 BaiduExporter 插件，在 Chrome 使用开发者模式进行安装；

缺点：速度不够迅雷快，稳定性欠佳。

you-get：流媒体下载。

说明：流媒体下载以往用的是硕鼠，但是硕鼠如今也不太稳定，而且毕竟软件比较大不太优雅。用 EagleGet 也是可以考虑的，但是有很多网站无法下载。所以 you-get 是一个很好的解决方案，小巧，灵活，支持多个网站的下载，功能强劲。

配置：

  • 一般来说需要安装 Python3 和 FFmpeg，但是 Windows 用户可以直接下载 exe安装包进行安装；
  • 解压缩后把文件夹放好，然后打开命令行工具cmd，cd去到 you-get 文件夹，然后输入you-get.bat（文件夹里面的 you-get.bat 起作用）即可进入 you-get 的下载界面；
  • 输入you-get link_of_video_or_picture，如 you-get http://www.fsf.org/blogs/rms/20140407-geneva-tedx-talk-free-software-free-society，就可以把视频下载到 you-get 文件夹里面；
  • 更具体的功能可以参考这篇文章。

缺点：命令行操作，有一定的门槛。

福昕阅读器 FoxitReader：PDF阅读器。

说明：免费，界面优美，功能完善。相比 Adobe Reader 又比较小巧；

配置：需要去除广告，“文件”——“偏好设置”——“常规”——把“显示广告条”的勾去掉；

缺点：软件比较大，资源消耗也比较大。如果没有特殊需求，可以直接用 Sumartra PDF，软件小，启动快，资源消耗小，免费，但功能少，颜值低。

PotPlayer：视频播放器。

说明：免费，解码功能强大，各项功能完善，颜值高；

配置：

  • 皮肤设置。下载 Zune皮肤并解压。打开 PotPlayer，“选项”——“基本”——“皮肤/配色”——“打开皮肤文件夹”，把Zune皮肤复制过来；
  • 打造无边框。点解重新搜索，然后选中Zune.dsf，勾选“视频下自动隐藏”和“以视频皮肤启动”，然后点击“皮肤/配色”下面的“皮肤输出方式/放大”，方式选为“使用Direct3D 9”，取消“不自动隐藏鼠标停留处的皮肤”选项。即可。
  • 有 V2EX 网友表示 Youtube 皮肤也很不错，配置方法和 Zune 皮肤一样；

缺点：视频播放器里面，属于没有槽点的一款软件。

网易云音乐：网络音频播放器。

说明：口碑之作，颜值及播放效果俱佳，网友评论有趣，歌单形式不错，推荐效果良好。可在微软官方商城下载 UWP 版（但桌面版调整白色图标比较轻松）；

配置：需要账号连接；

缺点：曲库经常因版权问题而缺失一些曲目。

Mcool：本地音频播放器。

说明：非常小巧，但已经足够；

配置：可设置为透明；

缺点：非常简陋，需要大量配置才能有较好的外观，需要熟悉快捷键才能流畅的使用。

输入法

输入法采用的是 Win10 自带的输入法。暂时使用良好，但不像搜狗这种支持多种功能，以及可自定义词库等等。不过搜狗输入法在高分屏上面表现很差，其它输入法也是（Bing 拼音在高分屏下表现还行，但是体验不好），所以有待继续观察。

5. 工作软件

Word/Excel/PPT：Office 三件套，不解释。

PPT 增强插件：

  • iSlider：提供了如统一字体等多个实用 PPT 工具；
  • icon8：离线图标。

Cmder：强化版控制台。

说明：Windows 原装的 Cmd 非常难用，无法自动补全，无法复制粘贴，无法高亮。因此可以采用 Cmder 来满足这些功能。Cmder 的完整版还与 Git for Windows 集成了。因此下载了 Cmder 完整版就相当于下载了 Git；

配置：无，具体可以定制化设置；

缺点：中文容易出 bug。

Anaconda：数据分析。

说明：集成了 Python 所需的大部分数据分析包，并且提供了数据分析界面如 Jupyter Notebook, JupyterLab, Spyder 等，基本是 Python 数据分析的标准软件了。而且 Anaconda 还提供了 R 的基础数据分析包以及 R 的数据分析新标准 tidyverse 包，然后再使用 Jupyter Notebook 来进行 R 的数据分析。因此可以不安装 RStudio 也可用 Ananconda 一起进行所有的数据分析；

配置：需要加入到系统路径中，下载可更换国内源，Jupyter Notebook 可下载插件；

缺点：不稳定，经常无法连接和下载。

Mendeley：文献管理。

说明：

  • Mendeley相对于其它文献管理软件，主要优势就是免费开源。其它的文献管理软件如Endnote、Citavi、ReadCube 等都是闭源或者收费，或者对文献数量有限制；
  • Mendeley 与 Endnote 相比，在文献管理的总逻辑上有不同，但具体在功能上差距不大，颜值上差一点；Citavi 独树一帜，非常严谨，还有知识总结的模式，用起来体验也非常不错，不过免费版文献数量受限；ReadCube 颜值最高，但在 Windows 系统下 bug 太多，严谨性也弱于 Citavi，因此不作推荐；
  • 相对于开源界的另一个重要的文献管理软件 Zotero，Mendeley 在 PDF 的管理和阅读上要明显高出一筹。但是在网页信息抓取，特别是中文文献，完全抓取失败，对中文 PDF 的支持也很差，无法识别；
  • 如果使用 Word 进行文献导入，Endnote 是支持最好的。因此如果是主要用 Word 写论文，那么理论上在 Mendeley 的基础上，还是要下载和使用 Endnote；

配置：需要账号连接，在 Chrome 中要安装插件抓取文献信息；

缺点：网页文献信息抓取能力弱，颜值低。事实上如果一款文献管理软件可以结合 Zotero 的网页抓取能力，Endnote 与 WOS 的集成能力，Citavi 的结构和管理方式，ReadCube 的颜值，以及免费无限量使用，那就是最好了。

FreeMind：思维导图。

说明：事实上，FreeMind 的模板很少，颜值低，被唾弃的多。但是一旦掌握少数的快捷键，FreeMind 就有快速产出思维导图的能力，所以我一直使用的是 FreeMind。但口碑更好的是 Xmind，因为我还没仔细使用，在此不做评价；

配置：无，需要熟记几个简单的快捷键；

缺点：模板少，颜值低，可定制能力差，只能自己看基本不能做展示。

Typora：Markdown 本地写作工具。

说明：Typora 让 Markdown 可以所见即所得，马上的渲染出最终页面，并且可以使用和 Word 相似的快捷键（如 Ctrl+B 是加粗）。颜值较为漂亮。如果进行博客写作是需要一个专门的 Markdown 写作工具的，Typora 在 Windows 系统下是个不错的选择；
配置：可在官网上下载一些特别的主题配色；
缺点：所见即所得的方式虽然方便，但有时操作性并不够源码好，并且有时候渲染出来的效果不一定非常好。

6. 网络工具

Inoreader：RSS 阅读器。

说明：在信息洪流中，RSS 是掌控个人信息获取的重要工具。特别是大量有益资源是来自于外文网站中，国内的公众号、头条、门户等信息源往往是二手资料，并且夹杂大量无益的信息。因此必须使用 RSS 阅读器去精选自己感兴趣的，高质量的信息来源。Inoreader 是较多人推荐的 RSS 阅读器，另外的同类产品还有 Feedly，Feedly 颜值较高，但在国内基本连不上，并且对订阅数量限制较多。因此 Inoreader 在国内更好用，并且 Inoreader 的手机端也设计得很不错，文章排版舒服，并且容易使用；

配置：在网上搜索高质量的与自身领域相关的国内外博客及资源站，添加到 Inoreader 里面；

缺点：Web 版的界面还是较为简陋，信息流的排版与 Feedly 相比欠佳。

Workflowy：笔记工具。

说明：Workflowy 可以记录特别结构化目录化的笔记，而且记录速度很快，非常适用于记录书籍。相似产品是幕布，幕布是中文的较为稳定，并且可以导出为思维导图；

配置：学习使用里面的快捷键来快速的创建条目以及在条目中前后跳跃，能极大提升记录效率。有 Chrome APP 版本，可离线使用；

缺点：插入图片功能不佳。免费版每月有条目的数量限制。

ProcessOn 和 Gliffy Diagram：流程图及基本图形制作。

说明：这两个软件可进行基本的图形绘制，可以认为是 Microsoft Visio 的低配版。但是这两个软件比较容易上手，方便操作。其中 ProcessOn 可以团队协作进行图形制作，但是有总图形数量限制。Gliffy Diagram 没有团队协作功能，但无数量限制。这两个软件都有 Chrome 应用版，可进行离线操作。

配置：无；

缺点：如上所述，一个有数量限制，一个无团队协作，视个人需求使用。

Trello：（个人）项目管理。

说明：Trello 采用了看板的模式，其初衷是进行事件管理，即项目管理。对于个人来说，往往可用于个人待办事项的整理，也可以作为个人资料知识的总结，还可以进行资料的整理例如学术期刊资料、学者资料等等；

配置：不同的使用目的具有不同的看板整理思路，需要有一定的分类和系统整理的思维，但也是在实践中不断摸索的；

缺点：国内连接不稳定，经常无法加载图片。

OverLeaf：LaTeX 编辑器。

说明：一般来说，编写中文 LaTeX 都是从 CTeX 上面下载安装包，然后本地安装后使用。但是云端的 LaTeX 编辑器就无需自己配置，同样可以编写中文，因此更为方便。原来云端 LaTeX 编辑器有 OverLeaf 和 ShareLaTeX 两款，后来两者合并，成为 Overleaf2。OverLeaf 颜值高，然而在国内稳定性非常差，速度特别慢。

配置：中文配置。

  • 如果直接编写中文的 LaTeX 文档，那么直接使用\documentclass{ctexart}即可；

  • 如果编写英文或者 Beamer（PPT）的 LaTeX 文档，中间要用到中文字符，那么\documentclass{beamer}后，使用\usepackage{xeCJK}；

  • 最后编译的时候应该选择XeLaTeX引擎。

缺点：非常不稳定，如果有重要的 LaTeX 写作需求还是使用本地软件更好，如 TeXLive。

7. 小软件

CopyTranslator：本地划词翻译。

说明：对于本地阅读外文 PDF 时，遇到不懂的单词短语往往要去到网站搜索。而本地划词翻译就可以直接解决这个问题；

配置：设置剪贴板监听，就可以复制文字的同时弹出翻译的结果；

缺点：颜值极低，稳定性不强容易崩溃，无法调整字体大小。

SciHub Desktop：利用 SciHub 下载论文。

说明：Sci-Hub 提供了大量论文的免费下载途径，但在网上经常打不开。一般人无法知道大量可用的 Sci-Hub，因此该桌面版就提供了一键式的 Sci-Hub 论文查询，非常方便；

配置：若使用 SciHub Desktop 下载论文，需要提供论文的 DOI 地址，可用该网址 输入论文题目来查询论文的 DOI；

缺点：Sci-Hub 源不稳定，因此该工具也无法解决这个问题。

微软必应词典：词典。

说明：微软商城的 UWP 版，界面漂亮，比起传统的 Lingos 和 Mdict，颜值实在是碾压。可以记录生词，可以全文翻译。

配置：无。

缺点：实用性相比于可以拓展词典的软件要弱。

Apowersoft：录屏。

说明：Apowersoft 口碑不错，免费，操作也容易，录制质量也好，无录制时长限制，无水印。虽然可替代产品非常多，但这款确实比其它好。

配置：无。

缺点：待考。

360 screener：截图小工具。

功能足够，速度比 Windows 原装的截图工具更快。若要功能更全面，应选择 FastStone Capture。

ProcessExplorer：更详细的任务管理器。

可以更细致的观察各个任务的进程。

HistCite Pro：文献引文网络分析。

可以分析通过 WOS 下载得到的领域论文，得出论文的被引频次，重要性等指标值，画出论文被引的网络图，从而识别论文的源流以及重要性。

Mathpix Snipping Tool：识别公式并转换为 LaTeX。

该工具可识别图片、PDF、网页里面的数学公式，并转换为 LaTeX (MathJax) 代码。以后再也不用手敲公式代码了。

PicGo：网络图床工具。

该软件提供便捷的上传图片到网络图床的途径，并且还提供了多个网络图床可供选择。
